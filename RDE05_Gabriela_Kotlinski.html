Hyperlinks

São uma referência dentro de um documento em hipertexto para outras partes desse mesmo documento, para outros, arquivos ou endereço eletrônico


Os links podem ser:
  • remotos
  • internos
  • locais
  • com endereço de correio
  • arquivos

Marcação de Hiperlinks

Acontece utilizando a tag "< a >" e a propriedade "href" que faz referência para o conteúdo desejado.


Sintaxe:

Hiperlinks Externos ou Remotos

São links que referenciam sites ou páginas da web.
Na propriedade "href" utilizamos a URL (Uniform Resource Locator) para a referência que desejamos.


Sintaxe:

Hiperlinks Internos

São links que apontam a um lugar diferente dentro da mesma página e possibilita acessar diferentes parágrafos ou seções quando o documento é extenso.


Sintaxe:

Hiperlinks Locais

São links que apontam para outras páginas dentro de um mesmo site>


Sintaxe:

Hiperlink em Imagens

É possível realizar hiperlinks em imagens envolvendo a tag de imagem com a tag de hiperlink.


Sintaxe: "< / a href = "referencia" > < img src = "imagem.jpg" > < / a> "

Hiperlinks para Downloads de Arquivos

São links que apontam para arquivos e possibilitam realizar downloads deste arquivo.

Hiperlinks para Endereço Eletrônico

São links que apontam para um endereço eletrônico.
