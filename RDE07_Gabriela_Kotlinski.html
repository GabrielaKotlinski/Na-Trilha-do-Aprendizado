Branches

São ramificações de um projeto de software ou documento e têm como objetivo o desenvolvimento paralelo de:

  • novas funcionalidades;
  • correção de problemas;
  • alteração de código.

Branch Master ou Main

braço principal de um projeto em um repositório;

Branch Secundária

braço secundário de um projeto em um repositório;

Observe os 2 casos:

Nesse caso o usuário é afetado, pois se o desenvolvedor realizar alterações e cometer erros estarão diretamente na branch principal;

Já nesse caso o usuário não é afetado, pois os commits são realizados em outras branches secundárias, não afetando diretamente a Branch Master.

Como criar uma Branch
