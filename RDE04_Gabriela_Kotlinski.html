IMAGENS

Representação visual de uma pessoa, objeto, paisagem.
Imagem Digital: é a representação de uma imagem bidimensional usando números binários codificados de modo a permitir seu armazenamento, transferência, impressãao ou reprodução, e seu processamento por meios eletrônicos

Formatos de Imagem

padrões de codificação, compressão e definição que determinam o tipo de arquivo gráfico, como por exemplo:

  • .BMP
  • .TIFF
  • .GIF
  • .SVG
  • .ICO
  • .JPEG
  • .PNG

Licenças e Repositórios de Imagens

Marcação de Imagens

As tags de imagens e seus parâmetros definem as imagens e suas propriedades.
As tags FIGURE e FIGCAPTION foram definidas a partir do HTML 5.

Marcação de Favicon

Define um ícone de favoritos que aparece na barra de títulos.

Sites:

  • icon archive
  • Favicon.cc
  • Favicon.io

Marcação de Listas

As tags de listas definem:

Lista Não-Ordenada
Define os itens em tópicos simples
Lista Ordenada
Define os itens em ordem com números (1,2,3...) ou letras(a,b,c...)
Lista Detalhada
Além de definir os itens, faz o detalhamento de cada um
