HTML

Hypertext Markup Language é uma linguagem de marcação de hipertexto baseada em tags, foi criada na década de 1990 por Tim Berners-Lee. Usado para desenvolver páginas e documentos eletrônicos para a internet, o HTML é responsável pelo conteúdo exibido em determinado site (texto,tabela,imagem,vídeo).

Tags

Definem ao navegador a marcação de conteúdo. Possuem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.
As tags podem possuir atributos que definem algum tipo de informação extra ao navegador.

Marcação de títulos

As tags de heading definem o título do texto e possuem 6 níveis de importância
O maior nível começa a partir do 1.

Marcação de parágrafos

As tags de parágrafo definem o início e fim de parágrafos no texto.

Marcação de quebra de linhas

As tags de quebra de linha produzem uma quebra no texto.
Úteis para escrever poemas, endereços e tópicos, onde a divisão de linha é importante.

Marcação para formatação de texto

As tags para formatação de texto definem destaque,tamanho de fonte, cor, etc.
